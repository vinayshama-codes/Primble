Key Architectural Properties
Property	Implementation
Stateless API	FastAPI with JWT вҖ” no server-side session
Async jobs	Queue + worker process вҖ” API never blocks on OCR
Multi-provider OCR	Single abstraction (ocr_service.py) вҖ” swap provider via env var
Schema-driven forms	JSON definitions drive extraction, validation, and PDF fill
Storage abstraction	s3_service.py speaks S3 API вҖ” works with AWS S3, R2, MinIO
Audit trail	Every sensitive action written to audit log via audit_service.py
Encryption at rest	crypto.py handles signatures and PII fields
